# 📌 Overview

LuxeManage is a command-line application designed to streamline hotel administration processes through a centralized database system.

The project demonstrates practical implementation of:

- Object-Oriented Programming
- JDBC Connectivity
- Relational Database Design
- CRUD Operations
- SQL Querying
- Persistent Data Management

---

# ✨ Features

### 🛏 Room Reservation System
Book rooms efficiently and manage reservations.

### 👥 Guest Management
Store and retrieve customer information.

### 💳 Billing Module
Generate invoices and calculate payments.

### 🏨 Room Allocation
Assign available rooms dynamically.

### 🔍 Search Functionality
Search records using reservation details.

### 🗄 Database Persistence
Maintain records securely in MySQL.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/VAIBHAVKHANDELWA/LuxeManage.git
```

## Import Database

Open MySQL and execute:

```sql
SOURCE hotel_backup.sql;
```

## Compile Program

```bash
javac HotelReservationSystem.java
```

## Run Program

```bash
java HotelReservationSystem
```
